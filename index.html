Lead Programmer. Technical Artist. UI Programmer and Designer.

Technical Stack:

Developed using Godot and written in GDScript. The art assets were downloaded from Itch.io.

Key Contributions:

I implemented the UI for the game, the power-up system, and the Smart Enemy AI. The Smart AI agent works by checking if the player character is inside its base or side of the map.
